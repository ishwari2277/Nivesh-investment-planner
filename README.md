Investment Prediction Tool 🚀

1. Overview
The Investment Prediction Tool is a web-based application that helps users estimate future investment growth based on their investment amount and risk tolerance. The tool divides investments into stocks and gold and predicts their value over a period of five years.

2. Features ✨
📊 Predicts investment growth based on selected risk tolerance.
🔄 Interactive form with dynamic result display.  
3. Installation and Setup ⚙️
1️⃣ Ensure that Python and Flask are installed on the system.
2️⃣ Install required dependencies by running:
pip install flask
3️⃣ Save the provided files in a new project folder.

4. Running the Application ▶️
1️⃣ Navigate to the project directory.
2️⃣ Run the Flask server using:
python app.py
3️⃣ Open a web browser and go to http://127.0.0.1:5000/ to access the tool.

5. Usage Instructions 📌
1️⃣ Enter the desired investment amount.
2️⃣ Select a risk tolerance level from the dropdown menu.
3️⃣ Click the "Predict" button to generate investment recommendations.
4️⃣ View stock and gold investment breakdown along with a visual pie chart.

6. Technologies Used 🛠
🌐 Frontend: HTML, CSS, JavaScript
🎨 Styling: Bootstrap for responsiveness
🖥 Backend: Flask for processing
7. Notes 📢
📈 The investment growth is calculated based on logical estimations and historical trends.
⚠️ The tool is intended for informational purposes and does not guarantee actual investment returns.
